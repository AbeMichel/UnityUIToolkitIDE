# Customizable Base IDE for UI Toolkit

A flexible code editor component for Unity UI Toolkit. This package provides a foundational IDE interface including syntax highlighting, text management, and extensible provider systems.

## Features

- Text document management with undo and redo history.
- Syntax highlighting using regex based tokenizers.
- Extensible autocomplete and symbol insight systems.
- Error reporting with tooltips and gutter markers.
- Customizable themes and editor configurations.
- Included support for C# and Python.

## Components

- Ide: The main VisualElement for the editor.
- TextDocument: Manages text content and history.
- EditorConfig: Controls visual settings and behavior.
- Providers: Interfaces for custom autocomplete, errors, and symbols.
